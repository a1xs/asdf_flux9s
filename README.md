# asdf-flux9s plugin for the  [asdf version manager](https://asdf-vm.com/)

A K9s-inspired terminal UI for monitoring Flux resources in real-time 

https://github.com/dgunzy/flux9s

## Install plugin
```bash
asdf plugin add flux9s https://github.com/a1xs/asdf_flux9s
```
#  Install app

## Install latest version
```bash
asdf install flux9s latest
```

## Install specific version
```bash
asdf install flux9s 0.8.1
```
